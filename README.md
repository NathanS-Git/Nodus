# Nodus

A node-based sandbox for digital logic.


The project contains an interactive graph that was inspired by the [CS Academy Graph Editor](https://csacademy.com/app/graph_editor/).

## Getting Started

### Prerequisites

- [.NET 7 SDK](https://dotnet.microsoft.com/download/dotnet/7.0) (Fable 4.24.0 is pinned to the .NET 7 SDK via `global.json` for stability on this project).
- [Node.js](https://nodejs.org/) with `npm`.

### Install dependencies

```bash
npm install
```

This also restores the local .NET tools (Fable, Femto) defined in `.config/dotnet-tools.json`.

### Build for production/static serving

```bash
npm run build
```

This compiles the F# sources with Fable and bundles them into `public/bundle.js`. The app can then be opened by serving `public/index.html` (for example, with the dev server below or any static file host).

### Start the development server

```bash
npm start
```

Then open [http://localhost:8080](http://localhost:8080) in your browser. The page hot-reloads when you edit the F# sources.

## Project Layout

```
├── src/
│   ├── Types.fs        # Domain types: Node, Edge, ToolMode, GraphState
│   ├── Physics.fs      # Force-directed simulation: repulsion, springs, damping
│   ├── GraphEditor.fs  # Graph operations, hit-testing, and canvas rendering
│   └── App.fs          # UI bootstrap, toolbar, event wiring, render loop
├── public/
│   ├── index.html      # Main HTML page
│   └── bundle.js       # Generated by Webpack
├── src/App.fsproj      # F# project file
├── package.json        # npm scripts and Node dependencies
├── webpack.config.js   # Webpack configuration
└── global.json         # Pins the .NET SDK version for Fable compatibility
```
